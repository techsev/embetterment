Auto-post progress progression to Main Twitter

Using a lot of what I learned on how to pull in data from twitter, I created a grunt task that posts my overall progress to my primary account.

First, I found a grunt plugin that reads in a CSS files and turns it into a json file that I imported into my grunt script. From there I found the overall percentage of my progress and used the same twitter api grunt plugin to post a quick update to my primary twitter account.
